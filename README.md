# 🪙 Coin Flip

A clean, animated coin-flip web app. No dependencies — pure HTML, CSS, and JavaScript.

## Features

- Animated 3D coin flip (heads = gold, tails = silver)
- Last 5 results displayed as colour-coded pills
- Running heads % and tails % tracker
- Text-to-speech button reads the most recent result aloud
- Spacebar / Enter also triggers a flip

## Getting Started

1. Clone or download this repo
2. Open `index.html` in your browser — no build step needed

## Project Structure

```
coinflip/
├── index.html   # Markup
├── style.css    # Styles & animations
└── script.js    # Logic & TTS
```

## Deploying to GitHub Pages

1. Push to a GitHub repo
2. Go to **Settings → Pages**
3. Set source to the `main` branch, root folder
4. Your app will be live at `https://<your-username>.github.io/<repo-name>/`
